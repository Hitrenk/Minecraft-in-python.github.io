Make a open source Minecraft!

This is the offical website of Minecraft-in-python. All of the source code are written in 100% pure python.

For someone who wants to write a client or mod: How to contribute to Minecraft-in-python reasonable.

Compare with Minecraft Java Edition and Bedrock Edition, we have the following advantages:

  • Open source
  • Register without web
  • Community support
  • Cross platform

Now the version of Minecraft is 0.3.2, which is still in develop. The following list is what this game should have in the future:

  • Entity(tnts)
  • Official modding api
  • Unlimited world
  • Game server
