Minecraft in python

This is the offical website of Minecraft-in-python.

The title means: Use python to write Minecraft. All of the source code are written in 100% pure python(except TK theme, GLSL).

All repository under our organization are open source under GPLv3 license. But, for the benefit of developers, the server is open source under MIT license.

Compare with Minecraft Java Edition and Bedrock Edition, Minecraft in python have the following advantages:

  • Open source
  • Register without web
  • Community support
  • Cross platform

Now the version of Minecraft is 0.2, which is still in develop. The following list is what this game should have in the future:

  • Useful GUI
  • Unlimited world
  • Entity
  • Server
  • ...
